# 51Talk 前端中台 - 抽奖活动配置手册
> 中台地址：https://act.51talk.com/admin/activity
> 权限问题：联系中央运营

---

## 一、活动概述
### 适用场景
提升出席率、付费率、用户活跃度

### 核心能力
- 支持**多语言适配**（按用户设备语言自动切换）
- 支持**实时到账**：香蕉币、次卡、兑换商城权益
- 支持**精准人群**：国家/地区、支付金额、支付时间、用户类型等
- 支持嵌入**活动中心/打卡中心**展示

### 奖品类型
次卡、香蕉币（积分）、实物、兑换商城权益

---

## 二、操作流程
### 1. 创建活动
- 入口：活动列表 → 新建活动 / 复制过往活动
- 复制活动可快速复用配置，节省编辑时间

#### 活动列表关键字段
- 活动ID、活动名称、用户类型、用户人群、活动类型
- 开始时间、结束时间、活动状态、操作（查看/修改/配置）

---

### 2. 设置用户可见性
入口：活动 → **修改**

#### 基础设置
- **活动名称**：仅后台识别，用户不可见
- **活动说明**：用户可见，**不支持多语言**，`<br/>` 换行
- **活动类型**：大转盘抽奖
- **海外属性**：非海外/海外/全部
- **是否启用**：上线前必须开启
- **头图**：
  - App：686×200
  - PC：按要求上传
- **活动时间**：用户仅可在此时间段访问

#### 展示位置
- 活动中心
- 首页Banner

#### 人群限制（勾选展示位置后生效）
- 学员剩余财富（user_asset）
- 首大单付款时间（first_buy_time）
- 首订单付费金额（first_order_money）
- 学科属性：全部/英语/普通话

---

### 3. 配置奖品（核心）
入口：活动 → **模块列表** → 抽奖模块 → **设置奖品**

#### 奖品管理字段
| 项 | 说明 |
| --- | --- |
| 奖品名称 | 页面展示名称 |
| 奖品类型 | 次卡/积分(香蕉币)/实物/权益 |
| 奖品总库存 | 该奖项可被抽中的总次数 |
| 单次中奖数量 | 抽中一次发放的数量 |
| 中奖概率 | 单位：%；概率=0或库存空则不中奖 |
| 层级概率 | 一般与中奖概率保持一致 |
| 关联ID/Code | 次卡/积分填Code；权益填ID+Code=act；实物不填 |
| 排序 | 数值小 → 转盘靠前位置 |
| 预览图 | Web/Wap 均需上传；尺寸 200×200 |
| 白名单 | 学员ID逗号分隔，为空则全员可抽 |
| 立即发放 | 虚拟奖品选“是”，实时到账 |
| 允许重复抽中 | 控制用户是否可重复中同一奖 |
| 是否启用 | 开启后可被抽中 |

#### 奖品配置要点
- Code/权益ID：向**中央运营**申请
- 额外费用：需提前审批
- 配置完成后：联系**技术（陈智元）**配置抽奖次数
  - 提供：活动ID + 单用户可抽次数

---

## 三、抽奖规则（示例）
- 活动对象：指定时间段、指定用户
- 到账规则：
  - 次卡：自动到账
  - 香蕉币：15分钟内添加
  - 海洋公园套票/年票：联系客服兑换

## 五、上线检查清单
1. 活动时间正确
2. 活动状态：**启用**
3. 奖品概率总和=100%
4. 奖品库存、数量、类型正确
5. 图片上传完整（App/PC/Web/Wap）
6. 人群限制配置正确
7. 联系技术配置**抽奖次数**
8. 虚拟奖品开启**立即发放**
