# Overseas-CC-Operations 📞
海外前端 CC 运营标准操作手册与配置文档库

## 文档说明
本仓库为 51Talk 海外前端 CC 运营团队提供**全流程操作指引、系统配置、数据监控、活动策略、线索流转**等标准化参考，用于日常工作执行、新人培训与流程对齐。

适用岗位：海外前端运营、CC 主管、活动配置、素材维护、数据监控
---

## 📖 项目简介
这是一项面向海外前端运营的AI技能。
安装后，您的AI助手可以为用户提供**线索分配、运营技巧、套餐设计与配置**等全套CC运营技能。我们希望通过将智能功能集成到前端用户服务中，帮助传统企业升级服务体验，提高运营效率，并为全球CC运营打造更便捷、更个性化的运营旅程。

---

## ✨ 核心功能
- 🧠 **智能分配指导**：自动解答注册/约课/出席/重登录等全场景线索分配规则。
- ⚙️ **配置全流程教学**：从筛选条件、优先级设置到CRM后台配置，手把手教学。
- 🛠️ **运营工具支持**：讲解排班、定级、抢单池、分配系数与任务上限等配套系统。
- 🐛 **问题排查助手**：快速定位优先级冲突、分配不生效、状态异常等常见问题。

## 核心模块
1. **产品与活动配置**
   - 前端套餐、勤学奖、优惠券配置
   - Banner/轮播图/弹窗配置
   - 抽奖活动配置与上线检查
2. **AC 教室与素材维护**
   - 视频关单素材上传、修改、发布
   - H5/PDF 教材维护
3. **CC 运营策略**
   - 产品运营、促销、激励、资源、转介绍策略
   - 高拉动手段与战役打法
4. **Leads 分配与生命周期**
   - 注册/约课/出席/重登录分配逻辑
   - 公海、小金库、掉落规则
5. **数据监控与分析**
   - 日/月核心指标监控
   - 转化漏斗、进量、转率、客单诊断
   - 市场/转介绍双渠道分析

## 使用方式
- 日常操作直接查阅对应 `.md`/`.txt` 文档
- 配置类文档按系统地址直接进入后台
- 策略类用于活动策划、激励设计、目标拆解
- 数据类用于日报、周报、问题定位

## 权限说明
- 部分系统需申请权限：赠课系统、CC 分配、抽奖后台
- 权限申请与技术对接按文档指定联系人执行
