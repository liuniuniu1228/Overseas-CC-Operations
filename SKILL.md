# Overseas CC Operations Skill
海外 CC 运营核心技能与实战能力清单

## 一、基础配置技能
### 1.1 套餐与权益配置
- 前端套餐配置：币种、金额、有效期、地区、可见组
- 优惠券配置：查询、新建、规则、有效期、上架状态
- 勤学奖配置：活动时间、状态、用户范围
- 赠课配置：官网赠课 / 活动赠课区分

### 1.2 素材与 Banner 配置
- APP/PC/WAP 轮播图尺寸与上传
- 首页弹窗（Popup）配置
- 多语言地区投放：简体/繁体/英语/泰语/阿拉伯语
- 权重、区域、应用类型、时间范围设置

### 1.3 AC 教室素材维护
- PDF 非 H5 教材上传
- H5 教材上传、生成 zip、发布
- 素材修改：删除旧页 → 重传 → 预览 → 发布
- AC 教室教材目录维护

### 1.4 抽奖活动配置
- 新建/复制活动、头图上传
- 奖品配置：次卡/香蕉币/实物/权益
- 概率、库存、立即发放、白名单设置
- 上线检查清单：时间、启用、概率、次数、到账

## 二、Leads 分配与运营技能
### 2.1 分配逻辑掌握
- 分配场景：注册/约课/出席/重登录
- 分配维度：国家、渠道、年龄、身份、尾号、等级
- 分配方式：平均分配 / 到人分配
- 优先级配置与生效规则

### 2.2 线索生命周期管理
- 市场线索 vs 转介绍线索分配规则
- Leads 掉落：未出席/未付费自动回公海
- 小金库使用：容量上限、不掉落规则
- 公海捞取与跟进规范

### 2.3 CC 团队管理技能
- 销售排班、分配周期配置
- 例子分配系数、日上限、抢单配置
- CC 定级（A/B/C/D）与系数联动
- 销售任务与奖励额度配置

## 三、运营策略技能
### 3.1 产品运营
- 大套餐下架、新品上线、爆款大单打造
- 涨价节奏、团购周期、边缘套餐回炉

### 3.2 促销运营
- 赠课：常规赠课、勤学奖 V1/V2、倒计时递减
- 保价、无忧购退费、加送请假次数
- 体验课后 48h 限时立减

### 3.3 激励运营
- 个人/小组/大组 PK、分层排名赛
- 大战役提点提升、双记业绩
- 团建奖励、吃喝奖励、旅游奖励
- 末位督促与落后组扶持

### 3.4 转介绍运营
- 基础奖励、合伙人额外奖励
- 朋友圈转码、出席送绘本
- 季度挑战与头部榜样打造

## 四、数据监控与分析技能
### 4.1 核心指标监控
- 注册、转率、成单、客单、营收
- 市场渠道 vs 转介绍渠道双维度
- 大区 → 小组 → 个人逐层拆解

### 4.2 问题诊断能力
- 注册不足 → 进量问题
- 转率低 → 转化/跟进/素材问题
- 客单低 → 套餐/策略问题

### 4.3 漏斗转化分析
- 注册→预约→出席→付费全链路
- 出席付费率、预约出席率、注册预约率
- 公海质量、历史 Leads 跟进效率

### 4.4 数据来源使用
- 通话数据、分配数据、体验课数据、订单数据
-  Lingyun Task 号与常用 SQL 场景

## 五、实战执行能力
### 5.1 活动全流程操盘
1. 方案确定 → 套餐/权益配置
2. 素材制作 → Banner/视频关单上传
3. 分配/激励配置 → 上线检查
4. 数据监控 → 问题调整 → 复盘

### 5.2 高转化打法
- 大节点倒计时赠课
- 月初开团、月末冲刺
- 老续费时光机、窗口期激励
- 转介绍高转化政策复用

### 5.3 风险控制
- 转介绍作弊防范
- 退费摊销与高退费月应对
- 配置上线前必查清单配
- 同一区域内所有到人分配共用一套系数与上限
