# Overseas CC&TMK Operations Skill
海外CC&TMK运营核心技能与实战能力清单（线索分配系统全流程）

## 一、基础配置技能
### 1.1 套餐与权益配置
- 前端套餐配置：多币种适配、金额梯度设计、有效期规则（自然日/工作日）、地区灰度投放、可见组精准配置
- 优惠券配置：多维度查询、新建规则（满减/直降/叠加限制）、有效期、上下架批量操作
- 勤学奖配置：活动时间、生效状态、用户范围（新客/老客/特定等级）
- 赠课配置：官网固定赠课 / 活动临时赠课、生效/失效规则、跨地区适配

### 1.2 素材与 Banner 配置
- APP/PC/WAP 轮播图：尺寸规范、格式校验、上传后缓存清理
- 首页弹窗（Popup）：触发规则、展示时长、关闭逻辑
- 多语言投放：简/繁/英/泰/阿语文案，阿拉伯语右对齐排版
- 权重、区域、应用类型、时间范围组合生效与优先级

### 1.3 AC 教室素材维护
- PDF 教材：分册上传、版本命名、地区可见性
- H5 教材：上传→zip→测试→发布，版本回滚
- 素材修改：删旧页→重传→预览→发布
- 教材目录：层级分类、多语言名称、更新同步

### 1.4 抽奖活动配置
- 新建/复制活动、头图适配、短链生成
- 奖品：次卡/香蕉币/实物/权益、发放规则
- 概率、库存、次数、白名单配置
- 上线检查：时间、状态、概率、到账测试

## 二、Leads 分配与运营技能
### 2.1 分配逻辑掌握
- 分配场景：注册/约课/出席/重登录/转介绍
- 分配维度：国家、渠道、年龄、身份、尾号、等级
- 分配方式：平均分配 / 到人分配 / 区域分配
- 优先级配置与冲突兜底规则

### 2.2 线索生命周期管理
- 市场线索 vs 转介绍线索：池隔离、分配规则区分
- Leads 掉落：未出席/未付费自动回公海
- 小金库：容量上限、不掉落规则
- 公海捞取、跟进时效、无效标注规范

### 2.3 CC 团队管理技能
- 销售排班、分配周期（时区适配）
- 分配系数、日上限、抢单池配置
- CC 定级 A/B/C/D 与系数联动
- 任务、阶梯提点、奖励额度配置

### 2.4 CRM 专项配置
- 线索自定义字段、必填项设置
- 跟进记录模板、标签体系
- 客户画像自动/手动打标
- 数据同步与异常告警

## 四、数据监控与分析技能
### 4.1 核心指标监控
- 注册、转化率、成单、客单、营收（多币种）
- 市场渠道 vs 转介绍渠道双维度
- 大区→小组→个人逐层拆解

### 4.2 问题诊断能力
- 注册不足：进量/渠道/链路问题
- 转率低：跟进时效/话术/素材/线索质量
- 客单低：套餐策略/销售主推方向

### 4.3 漏斗转化分析
- 注册→预约→出席→付费→复购全链路
- 注册预约率、预约出席率、出席付费率
- 公海质量、历史 Leads 跟进效率

### 4.4 数据来源使用
- 通话、分配、体验课、订单数据
- Lingyun Task 号、常用 SQL 场景
- 数据导出与可视化

## 六、工具与常用链接技能（新增）
### 6.1 新人专属 Referral Link 生成（国内CRM）
**Q：新人入职没有 referral link，怎么生成？**
A：登录国内CRM后，用固定链接替换参数生成。

**基础链接**
https://crm.51suyang.cn/admin/test/test.php

**生成规则**
- `ids=111111111` → 替换为**本人CRM ID**
- `type=cc` → CC岗位保留；**SS岗位改为 type=is**

**示例**
- CC：`https://crm.51suyang.cn/admin/test/test.php?ids=123456789&type=cc`
- SS：`https://crm.51suyang.cn/admin/test/test.php?ids=123456789&type=is`

**返回结果说明**
- `专属LandingPage已存在`：已生成成功，直接使用
- 参数错误：检查ID是否纯数字、type是否cc/is
- 无权限：先登录CRM再访问
