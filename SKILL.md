# Overseas CC&TMK Operations Skill
海外CC&TMK运营核心技能与实战能力清单（线索分配系统全流程）

## 一、基础配置技能
### 1.1 套餐与权益配置
- 前端套餐配置：多币种适配、金额梯度设计、有效期规则（按自然日/工作日区分）、地区灰度投放、可见组精准配置
- 优惠券配置：多维度查询（时间/类型/使用范围）、新建规则（满减/直降/叠加限制）、有效期精准控制、上下架状态批量操作
- 勤学奖配置：活动时间区间锁定、生效状态校验、用户范围（新客/老客/特定等级）精准圈选
- 赠课配置：官网固定赠课 vs 活动临时赠课区分配置、赠课课时生效/失效规则、跨地区赠课适配

### 1.2 素材与 Banner 配置
- APP/PC/WAP 轮播图：各端尺寸规范（附标准尺寸表）、格式校验（PNG/JPG/WEBP）、上传后缓存清理
- 首页弹窗（Popup）：触发规则（首次登录/间隔登录/特定地区）、展示时长、关闭逻辑配置
- 多语言地区投放：简体/繁体/英语/泰语/阿拉伯语文案适配、右对齐（阿拉伯语）排版调整
- 权重、区域、应用类型、时间范围：多条件组合生效、优先级覆盖规则

### 1.3 AC 教室素材维护
- PDF 非 H5 教材：分册上传、版本命名规范、地区可见性配置
- H5 教材：上传流程（源文件→zip打包→环境测试→正式发布）、版本回滚机制
- 素材修改：删除旧页→重传新页→预览校验（多端适配）→灰度发布→全量发布
- AC 教室教材目录：层级分类、多语言目录名称、教材更新通知同步

### 1.4 抽奖活动配置
- 活动基础配置：新建/复制活动模板、头图尺寸适配、活动短链生成
- 奖品配置：次卡（有效期叠加规则）、香蕉币（自动到账校验）、实物（物流对接）、权益（兑换码生成）
- 核心规则：概率精准配置（万分级）、库存实时监控、立即发放/延迟发放切换、白名单批量导入
- 上线检查清单：时间区间（时区适配）、启用状态、概率总和（≤100%）、参与次数限制、奖品到账测试

## 二、Leads 分配与运营技能
### 2.1 分配逻辑掌握
- 核心分配场景：注册/约课/出席/重登录/转介绍线索专项分配
- 精准分配维度：国家/渠道/年龄/身份/手机号尾号/用户等级/首次触达时间
- 分配方式：平均分配（负载均衡）、到人分配（指定销售）、区域分配（按大区/小国）
- 优先级配置：规则生效顺序（置顶/下移）、多条件冲突时的优先级兜底规则

### 2.2 线索生命周期管理
- 线索类型区分：市场线索（公海公共池）vs 转介绍线索（专属池）分配规则隔离
- Leads 掉落规则：未出席（24h后）/未付费（72h后）自动回公海、掉落前提醒跟进
- 小金库配置：容量上限（按销售等级调整）、不掉落规则（高意向线索锁定）
- 公海操作规范：捞取权限（等级/绩效关联）、捞取后跟进时效（2h内首次触达）、无效线索标注规则

### 2.3 CC 团队管理技能
- 基础配置：销售排班（时区适配）、分配周期（按小时/按天/按批次）
- 核心参数：分配系数（A/B/C/D级销售差异化）、日上限（防过载）、抢单池（开启/关闭/抢单时效）
- 销售定级联动：A/B/C/D级系数与线索分配量挂钩、降级/升级后系数自动同步
- 任务与奖励：销售日/周/月任务配置、奖励额度（阶梯式提点）、业绩核算规则

### 2.4 CRM 专项配置（新增）
- 线索字段配置：自定义字段（地区/渠道/意向套餐）、字段必填项设置
- 跟进记录模板：标准化跟进标签（首次沟通/需求确认/异议处理）、记录提交时效
- 客户画像标签：自动打标（基于行为）、手动打标（基于沟通）、标签联动分配规则
- 数据同步：线索分配数据与CRM实时同步、异常数据告警规则

## 三、运营策略技能
### 3.1 产品运营
- 套餐生命周期：大套餐下架（提前7天预告）、新品上线（灰度测试→全量）、爆款大单（权益叠加打造）
- 价格策略：涨价节奏（分地区梯度涨价）、团购周期（成团人数/时效配置）、边缘套餐回炉（权益升级复用）

### 3.2 促销运营
- 赠课策略：常规赠课（固定课时）、勤学奖 V1/V2（阶梯式赠课）、倒计时递减（越晚赠课越少）
- 保障类权益：保价（30天价格保护）、无忧购退费（退费规则公示）、加送请假次数（与套餐绑定）
- 限时转化：体验课后48h限时立减、跨时区时间精准计算、优惠到期前提醒

### 3.3 激励运营
- 内部激励：个人/小组/大组 PK 赛（规则/奖品/排名公示）、分层排名赛（按等级分组）
- 业绩激励：大战役提点提升（临时系数调整）、双记业绩（个人+团队）、阶梯式奖励
- 非物质激励：团建奖励、即时吃喝奖励、月度旅游奖励、标杆销售案例分享
- 兜底机制：末位督促（专项培训）、落后组扶持（线索倾斜+导师带教）

### 3.4 转介绍运营
- 奖励体系：基础奖励（固定课时/香蕉币）、合伙人额外奖励（阶梯式）
- 传播玩法：朋友圈转码（专属二维码生成）、出席送绘本（实物+电子版）
- 长效激励：季度挑战（高奖励门槛）、头部榜样打造（案例包装+经验复用）
- 作弊防范：转介绍关系校验、异常数据监控（同一设备/手机号/地址）

## 四、数据监控与分析技能
### 4.1 核心指标监控
- 核心大盘指标：注册量、转化率、成单量、客单价、营收（按币种/人民币双维度核算）
- 渠道维度：市场渠道（付费/自然）vs 转介绍渠道（老客推荐/合伙人）双维度对比
- 拆解逻辑：大区 → 小组 → 个人逐层拆解、异常指标定位（时间/地区/销售）

### 4.2 问题诊断能力
- 注册不足：进量渠道问题（投放/素材）、地区覆盖问题、注册链路卡顿
- 转率低：转化话术问题、跟进时效问题、素材匹配度问题、线索质量问题
- 客单低：套餐推荐策略问题、高客单权益不足、销售主推方向偏差

### 4.3 漏斗转化分析
- 全链路漏斗：注册→预约→出席→付费→复购全链路监控
- 关键转化率：出席付费率、预约出席率、注册预约率、复购率
- 线索效率：公海线索质量评分、历史Leads跟进效率（触达率/成单率）

### 4.4 数据来源使用
- 核心数据类型：通话数据（时长/接通率）、分配数据（量/效率）、体验课数据（出席/评价）、订单数据（金额/退费）
- 工具使用：Lingyun Task 号操作规范、常用SQL场景（线索明细/转化明细/退费明细）、数据导出与可视化

## 五、实战执行能力
### 5.1 活动全流程操盘
1. 方案确定：目标拆解→套餐/权益配置（多版本备选）
2. 素材制作：文案/设计→Banner/视频/关单素材上传（多端测试）
3. 配置上线：分配规则/激励规则配置→全维度上线检查（时间/权限/规则）
4. 过程监控：实时数据监控→问题快速调整（系数/素材/规则）→复盘沉淀（成功/失败案例）

### 5.2 高转化打法
- 节点营销：大节点（节日/季度末）倒计时赠课、跨时区节点时间适配
- 周期打法：月初开团（成团奖励）、月末冲刺（临时提点）
- 老客运营：老续费时光机（历史价格回溯）、窗口期激励（限时续费权益）
- 策略复用：转介绍高转化政策跨地区复用、成功案例快速复制

### 5.3 风险控制
- 作弊防范：转介绍作弊（设备/IP/手机号校验）、虚假订单识别
- 退费管理：退费摊销计算、高退费月应对策略（提前预警+挽回）
- 配置校验：上线前必查清单（规则/时间/权限/数据同步）、多人交叉核对
- 规则兜底：同一区域内所有到人分配共用一套系数与上限、配置变更留痕

### 5.4 应急处理
- 配置异常：线索分配错误→紧急暂停分配→数据回滚→规则修正→重新上线
- 素材问题：错误素材上线→紧急下架→替换素材→缓存清理→用户侧验证
- 数据异常：注册/成单暴跌→快速定位（渠道/分配/素材）→临时策略调整
